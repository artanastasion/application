# Redactor 
redactor using [Electron](https://github.com/electron/electron) and [Vue.js](https://github.com/vuejs/vue)

# Installation 
Before you begin, make sure that Node.js is installed.
To make sure that it is installed correctly run:

` 
npm -v
node -v
`
To install app and all dependencies run:

git clone https://github.com/artanastasion/application
cd application
npm install

# Using
To start app run:
` npm run electron:serve`
